# SQL-Retail-Sales-Analysis

## Project Overview
This project is developed using SQL to manage retail store sales data. It helps track products, orders, quantity sold, and revenue generated.

## Table Used
Sales

## Features
- Store sales records
- Calculate total revenue
- Calculate total quantity sold
- Count total orders
- Find category-wise revenue
- Calculate average product price

## SQL Concepts Used
- CREATE TABLE
- INSERT INTO
- SELECT
- SUM()
- COUNT()
- AVG()
- GROUP BY

## Queries Performed
1. Total Revenue
2. Total Quantity Sold
3. Total Orders
4. Category-wise Revenue
5. Average Product Price
